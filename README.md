
# Credit Card Approval Prediction
## Abstract
Credit Card Approval Prediction is a machine learning classification project that predicts whether a credit card application will be approved or rejected based on the applicant's demographic and financial history. The project combines a trained ML model with a simple web interface, allowing users to enter their details and instantly view a prediction result.

## Problem Statement
Banks and financial institutions receive a large number of credit card applications daily. Manually reviewing each application is time-consuming and prone to human bias. This project aims to automate the credit card approval decision using historical applicant data, helping to make the process faster, more consistent, and data-driven.

## Features
- Predicts credit card approval status (Approved/Rejected) based on user input
- Data cleaning and pre-processing pipeline for handling missing values and outliers
- Exploratory data analysis and visualization of key features
- Machine learning model trained and evaluated on real-world credit card data
- Simple, responsive web interface for entering applicant details
- Real-time prediction through a Flask backend

## Tech Stack
- **Language:** Python 3.8+
- **Machine Learning:** scikit-learn, imbalanced-learn
- **Data Analysis:** pandas, numpy
- **Visualization:** matplotlib, seaborn
- **Backend:** Flask (app.py)
- **Frontend:** HTML, CSS, JavaScript
- **Development Environment:** Jupyter Notebook / Anaconda Navigator

## Installation Steps
1. Clone the repository:
   git clone
    https://github.com/yennihema/Credit-Card-Approval-Prediction.git
cd Credit-Card-Approval-Prediction
2. Install the required dependencies:
 pip install -r requirements.txt
 3. Open the web application in your browser (usually at `http://127.0.0.1:5000/`).
4. Enter the applicant details in the form and submit to view the prediction result.
